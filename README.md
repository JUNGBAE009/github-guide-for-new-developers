# Git과 Github 의 차이점

<br>

## Git = 로컬(맥북,크롬북,데스크탑...) 버전 관리 시스템
#### 시간에 따라 파일 변경사항을 추적하고 기록 해주는 소프트웨어 시스템
#### (타인이 나의 작업 내용을 알 수 없음)

## Github = 온라인 코드 공유
#### 클라우드 서비스라 타인과
#### 소스코드 공유가 가능하다.
#### (타인과 작업 내용을 공유 가능)
